This is my attempt at conquering the Bayesian statistics coursework in my Master's curriculum. 
I have included explanations & clarifications wherever they felt necessary. 

Note that, this academic endeavour only addresses the exercise problems that actually require coded solutions. 
Here is a [link to my handwritten notes](https://drive.google.com/file/d/1CE16Hu46E8y0of6WP7IagFOVoXBkcNwU/view?usp=sharing). I apologize if they appear a bit "technologically challenged". 
They're literally handwritten.

The data used for these problems can be found on the [author's (Peter D. Hoff's) website](https://www2.stat.duke.edu/~pdh10/FCBS/Exercises/).
In some application parts, I was unable to download the relevant data. 
In such cases, I was forced to copy-paste or I took screenshots and told AI to transcribe the data.

I applied set.seed() whenever randomness was involved. 
So, you should have no problem with reproducing the results. 

However, I didn't write any code to tackle the inline examples or replicate numerical figures and graphs within the textbook. 
You can find the .R files for them on the author's website. 
I recommend trying them on your own first.

That being said, if you find the code to be faulty or sketchy then please let me know in the comments.
